We Split

WeSplit is a web app which allows users to organize their bills easily. WeSplit allows users to create their own groups and then add different bills to each group which the group members can access.
Users can then settle their debt on the website by entering how much they paid each time. Each user also has an account, allowing a user's activity to be easily stored each time.

Aim Haskell

Aim Haskell is an aim trainer, primarily made using the gloss library. The game emulates an aim trainer's core feature, allowing the user to train different aspects of their aim from precision to flicking.
The game also includes a scoring system where if a user manages to achieve a high score then they can save their score along with their name.

Risk Predictor

This project was part of the CS261 Software Engineering module, sponsored by Deutsche Bank. Developed by a team of 6, the aim was to create a web-based project monitoring system for companies.
Our solution allowed users to create teams, projects, and tasks while monitoring project risks. The application dynamically updated risk levels based on factors like team size, budget, and deadline.
This project enhanced my web development skills and provided insights into group collaboration in software engineering projects.

Parser and Evaluator

Created a parser and a lexer using JavaCC that is able to recognize programs of a created language. Also extended the parser such that it is able to evaluate the output of programs given in the created language.
An example program input would be:
DEF MAIN { ABCD(1) } ;
DEF ABCD xyz { BCD(xyz) } ;
DEF BCD xy { 2*CD(xy) } ;
DEF CD x { D(x)+EF(x) } ;
DEF D x { 10 } ;
DEF EF x { 10*x } ;
 With the output being PASS 40
